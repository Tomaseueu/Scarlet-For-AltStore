# Scarlet For AltStore

![AltStore](https://img.shields.io/badge/AltStore-Classic-blue)
![iOS](https://img.shields.io/badge/iOS-Compatible-black)
![Status](https://img.shields.io/badge/Status-Active-green)

A custom **AltStore Classic source** for Scarlet IPA.

This repository provides an AltStore source (`scarlet-for-altstore.json`) that allows users to add Scarlet to AltStore Classic and install it directly from the AltStore app.

---

## 📱 About

**Scarlet For AltStore** is an AltStore Classic repository containing Scarlet.

Features:

- ✅ AltStore Classic compatible
- ✅ Simple JSON source
- ✅ Easy installation
- ✅ Automatic updates through AltStore

---

## 🔗 Add Source to AltStore

Copy this URL:

```
https://raw.githubusercontent.com/Tomaseueu/Scarlet-For-AltStore/main/scarlet-for-altstore.json
```

### Steps:

1. Open **AltStore Classic**
2. Go to **Sources**
3. Press **+**
4. Paste the URL
5. Press **Add Source**
6. Install Scarlet
